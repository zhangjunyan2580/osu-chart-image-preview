An application for previewing osu! beatmaps. It will copy the generated image to the clipboard, feel free to share it to your friends.

<img style="width: 25%; height: auto;" alt="image" src="https://github.com/user-attachments/assets/f9db8122-43e3-4c54-add1-8d511add5474" />

Features:
- [x] Preview taiko specific beatmaps
- [ ] Preview catch specific beatmaps
- [x] Preview mania specific beatmaps
- [ ] Preview taiko converted beatmaps
- [ ] Preview catch converted beatmaps
- [ ] Preview mania converted beatmaps
- [x] Use local beatmap files (.osu)
- [x] Use online beatmaps (by beatmap ID)
